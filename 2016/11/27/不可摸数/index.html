不可摸数 杭电 1999

发表于 2016-11-27   |   阅读 次

题目链接

Problem Description

s(n)是正整数n的真因子之和，即小于n且整除n的因子和.例如s(12)=1+2+3+4+6=16.如果任何
数m，s(m)都不等于n,则称n为不可摸数.

Input

包含多组数据，首先输入T,表示有T组数据.每组数据1行给出n(2<=n<=1000)是整数。
Output如果n是不可摸数，输出yes，否则输出no
Sample Input`

3
2
5
8Sample Output

yes
yes
no`
