题目分析：

要求给出的一组数据中任意的三个数不能构成”3-free”(即三个数不能够是公差大于零的等差数列)，为了方便查找我们可以先把给出的这组数据按照升序排列，然后循环找出首项和末项，再判断他们的中间项是不是在这组数据中。

找中间项时如果还用for循环查找的话，相当于三层for循环嵌套，所需要的运行时间太大，可以用set容器或则map容器优化这个查找过程。
